About the Project

This project aims to deliver a functional, efficient, and impactful solution for automated waste classification using deep learning. Manual waste sorting is slow and prone to error; our system provides a meaningful solution through AI.

Methodology

  • Model: CNN (EfficientNetB3) using Transfer Learning
  • Dataset: TrashNet standard dataset structure (14,000+ images)
  • Optimization: Adam Optimizer with Label Smoothing
  • Preprocessing: 300x300 Resizing & Normalization (OpenCV)

Project Objectives

As per the FYP Proposal, our objectives include:

  • Designing an automated system for sustainable waste management.
  • Implementing data augmentation to ensure model reliability.
  • Developing a web interface for real-time user interaction.
  • Supporting digital transformation in environmental conservation.
